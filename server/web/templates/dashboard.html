Log File Storage

Log files are written by individual setup.sh scripts to the tool's install directory. Each successful install writes an INSTALL_RECEIPT.txt and a timestamped log under /tmp/airgap-cpp-devkit/logs/.

Click View in Terminal inside any tool card's Console section to see its log in the bottom drawer.
