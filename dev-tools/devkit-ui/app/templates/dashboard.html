Running as Administrator / Root
By default DevKit installs tools into your user profile. To install system-wide (accessible to all users), launch the devkit with elevated privileges. The install prefix switches automatically.
Windows (Git Bash)
Option A — Right-click Git Bash in the Start Menu → Run as administrator, then navigate and launch:
cd /c/path/to/airgap-cpp-devkit bash launch.sh
Option B — Right-click launch.sh → Open with Git Bash, then accept the UAC prompt.
✓ Tools install to C:\Program Files\airgap-cpp-devkit\
Linux (RHEL 8 / sudo)
Option A — Prefix with sudo:
sudo bash launch.sh
Option B — Switch to root first:
sudo su - cd /path/to/airgap-cpp-devkit bash launch.sh
✓ Tools install to /opt/airgap-cpp-devkit/
